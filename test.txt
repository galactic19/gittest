git 명령어 모음

git init ( 최초 초기화 해주는 명령어)

git add 파일명 ( 여러개의 전체 파일중 버젼 관리가 필요한 파일만 선별하여 commit 하도록 후보군들을 추가 시켜주는 작업)

git commit -m "메세지" (커밋하는 명령어 이고 간단하게 작성할떄 사용)

git commit -am  "커밋메세지"
(위 명령어는 add 와 commit를 동시에 할 수 있게 해주는데 한번이라도 commit 를 해준 대상에만 적용됨)

git status (상태확인)

git log (깃의 상태를 출력?)
